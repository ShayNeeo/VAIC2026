## Kết luận audit

Nền tảng thật hơn một demo trần trụi (circuit breaker, evidence
re-verification, injection screening đều là code thật, có test), nhưng
"hybrid retrieval" hiện tại là fusion tuyến tính của sparse thô + dense giả
lập bằng hash — **không phải** hybrid retrieval có RRF/reranker/embedding
ngữ nghĩa thật như prompt mô tả. `GroundingPack` đã được thiết kế nhưng
chưa từng chạy. Operations Agent hoàn toàn ngoài retrieval pipeline. Đây
chính là các điểm Phase 0-2 phải xử lý trước khi làm gì "nâng cao" hơn.
